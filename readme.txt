== Description ==

The AppAware Top Android Apps Widget shows the top Android apps among the AppAware community. The configuration possibilities are endless - you have the choice to select list types - "popular", "trending", "country-specific", "paid" etc. as well as choose particular category or timeframe.

To see the widget in action here: http://appaware.com/widgets/

To find more Android widgets for Wordpress visit http://appaware.com/widgets or http://playboard.me/widgets

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `appaware_top_apps_widget.php` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Add the widget to your Main Sidebar by clicking on Appearance->Widgets and drag-and-droping it on the Main Sidebar. The widget's configuration form appears and you can start using it on your blog.
